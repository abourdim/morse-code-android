[v1.0] — 2025-02-22

Added — Initial Release

Core Modules

  • 📚 Learn Tab — Interactive Morse code reference chart (A–Z, 0–9), click-to-hear audio playback
  • ✏️ Encode Tab — Text → Morse encoder with real-time conversion, visual lamp flasher, audio playback, speed control (WPM), copy and BLE send
  • 🔍 Decode Tab — Morse → Text decoder with live conversion, audio playback, hint for input format
  • 👆 Tap Lab — Interactive tap button (short=dot, long=dash), auto letter/word detection, waveform visualization, decoded text output
  • 🔊 Listen Station — Audio playback with adjustable frequency (400–1000 Hz) and speed (5–30 WPM), visual timeline with dot/dash indicators
  • 🤖 micro:bit Lab — Real BLE connection via Web Bluetooth API, SVG micro:bit simulator with 5×5 LED matrix, send/receive Morse via Nordic UART Service, MicroPython firmware code with copy button
  • 🏆 Challenge Zone — 4 game modes (Letter Quiz, Word Decode, Speed Round, SOS Challenge), scoring system with streak counter, local high score persistence
  • ❓ Help Tab — Morse glossary, learning path (6 steps), security & ethics section, troubleshooting guide

Platform Features

  • Trilingual UI — English, Français, العربية with full RTL support
  • 6 Themes — Mosque Gold, Zellige Blue, Andalus Green, Space Explorer, Jungle Quest, Robot Lab
  • 3 Skill Levels — 🟢 Newb (analogies), 🔵 Explorer (technical), 🟣 Developer (architecture)
  • Bismillah header — discreet, centered, calligraphy-inspired with ornaments
  • Activity Log — timestamped event log with clear/copy functionality
  • Toast Notifications — success/error/info toasts
  • Status Pill — BLE connection status indicator in header
  • Web Audio API — OscillatorNode-based Morse beep synthesis
  • BLE Architecture — Nordic UART Service (NUS), chunked TX (20-byte MTU), RX notifications, auto-reconnect (3 attempts)
  • CSS Theme System — 6 themes via CSS custom properties on [data-theme]
  • Privacy-first — no server, no tracking, no cookies, all local
  • Responsive design — mobile-friendly layout

micro:bit Integration

  • Web Bluetooth pairing with BBC micro:bit name prefix filter
  • UART Service UUID: 6e400001-b5a3-f393-e0a9-e50e24dcca9e
  • TX characteristic (notify): 6e400003-...
  • RX characteristic (write): 6e400002-...
  • Two-way communication: app sends text Morse, micro:bit sends button taps back
  • SVG simulator with clickable A/B buttons and LED matrix
  • MicroPython firmware provided in-app with copy button

Version Naming Convention

Package format: morse-kids-lab-v{MAJOR}.{MINOR}.zip

  • MAJOR — Breaking changes or complete rewrites
  • MINOR — New features, improvements, or fixes
