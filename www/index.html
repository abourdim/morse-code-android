📚 Morse Code Reference
Click any character to hear it!
🟢 Newb

Morse Code is like a secret flashlight language! You use only TWO things: short blinks (dots ·) and long blinks (dashes —). Every letter has its own pattern. SOS is ··· ——— ··· and it saved thousands of lives at sea!

🔵 Explorer

Morse Code was invented by Samuel Morse in 1836 for telegraph communication. Each character is encoded as a sequence of dits (·) and dahs (—). Timing is everything: a dit is 1 unit, a dah is 3 units, space between parts of a letter is 1 unit, between letters is 3 units, between words is 7 units. Standard speed is measured in WPM (Words Per Minute).

🟣 Developer

Morse is a variable-length binary encoding — similar to Huffman coding. Frequent letters (E=·, T=—) get shorter codes. With only 2 symbols, it maps to a binary tree where left=dit, right=dah. This makes it a prefix-free code: no sequence is a prefix of another when timing is used. The ITU-R M.1677 standard defines international Morse. Signal-to-noise characteristics make it detectable even 10dB below noise floor — better than voice or digital modes.

🤖 micro:bit Lab
Connect & communicate via Bluetooth
🟢 Newb

Your micro:bit is a tiny computer that can talk to this app using Bluetooth — like a secret walkie-talkie! Press Connect, then send Morse messages and watch your micro:bit buzz and flash!

🔵 Explorer

BLE (Bluetooth Low Energy) uses the Nordic UART Service (NUS) to create a serial-like connection. The app sends dots/dashes as text characters. The micro:bit receives them via UART and controls buzzer + LEDs accordingly. Range: ~10m — that's physical security!

🟣 Developer

Web Bluetooth API exposes GATT services. NUS UUIDs: Service 6E400001-..., TX (notify) 6E400003-..., RX (write) 6E400002-.... Data is UTF-8 over a 20-byte MTU. The micro:bit firmware parses single chars: '.'=dit beep (200ms), '-'=dah beep (600ms), ' '=letter gap, '/'=word gap. Two-way: buttons A/B on micro:bit send '.' and '-' back via TX characteristic.

🐍 micro:bit Firmware (MicroPython) 
from microbit import *
import music

uart.init(baudrate=115200)
DOT_MS  = 200
DASH_MS = 600
FREQ    = 800

while True:
    # RECEIVE from app → beep + flash
    if uart.any():
        ch = uart.read(1)
        if ch:
            c = chr(ch[0])
            if c == '.':
                display.show(Image.DIAMOND_SMALL)
                music.pitch(FREQ, DOT_MS)
                display.clear()
            elif c == '-':
                display.show(Image.DIAMOND)
                music.pitch(FREQ, DASH_MS)
                display.clear()
            elif c == ' ':
                sleep(DASH_MS)
            elif c == '/':
                sleep(DASH_MS * 2)
            sleep(100)

    # SEND to app → button press
    if button_a.was_pressed():
        uart.write('.')
        display.show(Image.DIAMOND_SMALL)
        sleep(150)
        display.clear()
    if button_b.was_pressed():
        uart.write('-')
        display.show(Image.DIAMOND)
        sleep(150)
        display.clear()

    sleep(30)

🔒 Security & Ethics

Morse Code is encoding, NOT encryption — anyone who knows the code can read your message! This is an important cybersecurity concept. In the old days, militaries added extra ciphers on top of Morse to keep messages secret. BLE is short-range (~10m), which provides physical security. Always communicate respectfully, even in code!

🔧 Troubleshooting

If BLE won't connect: Make sure you use Chrome/Edge browser. Enable Bluetooth on your device. Flash the MicroPython firmware to your micro:bit. If audio won't play: Click anywhere on the page first (browsers require user gesture for audio). Check that your device volume is on.
