📡 Morse Kids Lab — v1.2

بِسْمِ ٱللَّٰهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ

A fun, educational web app that teaches Morse Code to kids (ages 8–15) with real micro:bit BLE connectivity, audio synthesis, interactive challenges, and multi-language support.

Workshop DIY · Privacy-first · Local-first · Islam-friendly

✨ Features

Module Description
📚 Learn Interactive Morse alphabet chart — click any letter to hear it
✏️ Encode Text → Morse encoder with visual lamp flasher and audio
🔍 Decode Morse → Text decoder with live conversion
👆 Tap Lab Tap/press button to send Morse (short=dot, long=dash) with waveform
🔊 Listen Audio playback with frequency/speed controls and visual timeline
🤖 micro:bit Lab Real BLE connection + SVG simulator with 5×5 LED matrix
🏆 Challenge 4 game modes: Letter Quiz, Word Decode, Speed Round, SOS Challenge
❓ Help Glossary, learning path, security/ethics, troubleshooting, floating help button

Core Platform Features

  • 🌍 Trilingual — English, Français, العربية (with full RTL support)
  • 🎨 6 Themes — Mosque Gold, Zellige Blue, Andalus Green, Space, Jungle, Robot
  • 🎓 3 Skill Levels — 🟢 Newb · 🔵 Explorer · 🟣 Developer
  • 📡 Real BLE — Web Bluetooth connection to BBC micro:bit (Nordic UART Service)
  • 🔊 Web Audio API — Real-time Morse beep generation with adjustable pitch/speed
  • 🔒 Privacy-first — No server, no tracking, no cookies, 100% local
  • ⌨️ Keyboard Shortcuts — 1–8 tabs, Space BLE toggle, H help, Esc close

🚀 Quick Start

1. No Installation Required (Simplest)

Just open index.html in a modern browser. That's it!

# Double-click index.html
# OR open via terminal:
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows

2. Local Server (Recommended for BLE)

Web Bluetooth requires HTTPS or localhost:

# Python
python3 -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000

Then visit: http://localhost:8000

3. micro:bit Setup (Optional)

  1. Connect your BBC micro:bit V2 via USB
  2. Copy the MicroPython firmware from the micro:bit tab in the app
  3. Flash it using python.microbit.org
  4. Open the app in Chrome/Edge → micro:bit tab → Click Connect

🔒 Security & Ethics

  • Morse is encoding, NOT encryption — the app teaches this critical cybersecurity distinction
  • BLE is short-range (~10m) — teaches physical security concepts
  • Browser consent — Web Bluetooth always asks permission before connecting
  • No data transmitted — everything stays on your device
  • Respectful communication — even in code!
